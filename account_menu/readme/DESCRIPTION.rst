This module adds all missing menu entries for the **Account** module.

* Account Types
* Account Tags
* Account Groups
* Account Tax Groups
* Bank Statements
* Cash Registers
* Chart of Account Templates
* Account Templates
* Tax Templates
* Fiscal Position Templates

Additionally, this module also enables the option to enable or disable
Anglo-Saxon accounting in the Chart of Account Template form view and
in the Invoicing Settings.

Also contains the missing chart template field on the tax template form
(see https://github.com/odoo/odoo/pull/70956)
